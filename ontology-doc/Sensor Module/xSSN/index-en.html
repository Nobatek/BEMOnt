Ontology Specification Draft

Abstract

This ontology is based on BOT for describing building topology and extends it towards representing a wise set of building types (including residential and non residential buildings), spaces and storeys based on the Omniclass standard.

accuracydp back to ToC or Data Property ToC

IRI: http://www.bemserver.org/sensor/xssn/#accuracy

Accuracy represnts the closeness of agreement between the value of an observation and the true value of the observed quality.

frequencydp back to ToC or Data Property ToC

IRI: http://www.bemserver.org/sensor/xssn/#frequency

Frequency represents the frequency of an observation capture in ms.

is ambientdp back to ToC or Data Property ToC

IRI: http://www.bemserver.org/sensor/xssn/#isAmbient

isAmbient indicates whether the measure is an ambient one or not. For instance, a sensor measurement performed on the temperature in ome room/area of a building. Set points and measures made on HVAC equipment must not be ambient.

is index valuedp back to ToC or Data Property ToC

IRI: http://www.bemserver.org/sensor/xssn/#isIndexValue

isIndexValue indicates whether the measure is made on an index. In this case, values are cumulative.

is set pointdp back to ToC or Data Property ToC

IRI: http://www.bemserver.org/sensor/xssn/#isSetPoint

isSetPoint indicates whether the measure is a set point.

latencydp back to ToC or Data Property ToC

IRI: http://www.bemserver.org/sensor/xssn/#latency

Latency represents the time (in ms) between a request for an observation and the sensor providing a result.

max valuedp back to ToC or Data Property ToC

IRI: http://www.bemserver.org/sensor/xssn/#maxValue

minValue represents the maximum value accepted by the sensor

min valuedp back to ToC or Data Property ToC

IRI: http://www.bemserver.org/sensor/xssn/#minValue

minValue represents the minimum value accepted by the sensor

precisiondp back to ToC or Data Property ToC

IRI: http://www.bemserver.org/sensor/xssn/#precision

Precision represents the closeness of agreement between replicate observations on an unchanged or similar quality value: i.e., a measure of a sensor's ability to consitently reproduce an observation.

sensitivitydp back to ToC or Data Property ToC

IRI: http://www.bemserver.org/sensor/xssn/#sensitivity

Sensitivity represents the quotient of the change in a result of sensor and the corresponding change in a value of a quality being observed.

_Featurec back to ToC or Class ToC

IRI: http://www.opengis.net/gml/_Feature

_Feature is taken from the GeoRSS Feature Model. The model provides a general feature property which can be used to characterize any appropriate content as a geographic feature. Specific subproperties such as <where> associate the discerned feature with one of a limited number of geometry types which provide a numerical representation for analysis and visualization. Other subproperties describe additional commonly used feature attributes such as feature name and feature type.
Is defined by
http://www.opengis/net/gml/_Feature
has super-classes
located in hemisphere op only Hemisphere c
where op only _Geometry c
has sub-classes
FeatureOfInterest c

_Geometryc back to ToC or Class ToC

IRI: http://www.opengis.net/gml/_Geometry

GeoRSS feature model allows the description of rectangles, points, lines, and polygons as geometric representation properties of discerned geographic features. Simple Features' geometric representations consist of point and multipoint, curve (a.k.a. line) and multicurve, surface (polygon) and multisurface.
Is defined by
http://www.opengis/net/gml/_Geometry

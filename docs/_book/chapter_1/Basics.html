Contribution Basics

For those contributors in the documentation, it cloud be done:

  • Directly from Github and making commits. Then, automatically the repository is linked with a continous integration tool (CircleCI) that automatically update the gh-pages branch and the visualization

  • Making a clone of the repository (using for example Github Desktop tool) an then, make your changes in a different branch that lately has to be merged with the master branch. Once in the master branch is detected a change, "gh-pages" is directly updated.
